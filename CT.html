Morning Meeting

Every morning we read articles about technology online and create
a summary on it. We would then share what we learned with each other.

Crash Course

After the morning meeting and some lessons, we would watch a video from
Crash course to learn about computers and how they work. We would then
answer some questions to show our understanding of the topic.

Subnetting

Subnetting can be used to find information about a network using
some values such as a subnet mask and an IP address(optional).

Packet Tracer

We learned about packet tracers which can be used to set up
a network and see if it would work when set up in real life.

Virtual Machines

Virtual machines are operating systems that can be run within
another operating system through a software called hypervisor.

Netacad

Netacad is a software that offers many courses about
technology that you can learn about, some of them for free.

Network Types

There are many network types that connect many devices. Some of the
most common ones are Local Area Networks, Wide Area Networks,
Personal Area Networks, and Wireless Local Area Networks.

Python

Python is a high-level programming language that
can be used to make programs and applications.

HTML

HTML, also known as Hyper Text Markup Langauge,
is a programming language used to create and
design websites. This was also made using HTML.

Hexadecimal

Hexadecimal is a base 16 number system. It is mainly used in
IPv6 addresses and MAC addresses. It is more efficient than binary,
a base 2 number system, and may not run out of values for a long time.
