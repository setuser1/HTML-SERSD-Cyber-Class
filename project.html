Python

Python is a basic entry-level programming language which is slow compared to many other languages. It is mainly used for developing websites and software. It is very simple to use but can also be used for complex programs.

Packet Tracer

Packet Tracer is Cisco's simulation of switches. It is useful when learning subnetting and how to use switches. We can use this to learn how to set up networks and switches and other similar things without fear or risk of failure or damaging hardware.

Subnetting

Subnetting is the way of separating and specifying subnetworks. It allows us to keep networks organized and is an essential part of modern networking. This is one of the many basic skills you need in networking.
